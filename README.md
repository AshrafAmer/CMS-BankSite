# PHP Bank Site

Build a Bank Site using PHP and MySQL Languages

## Installation
To run this Site on your Localhost You must have:
* PHP7 installed.
* MySQL.
* Build a globe_bank DataBase.
* Build all needed tables.

## Usage
Use Wampserver to access this files to your localhost.
